# 🎧 AetherTune

**Right Music Every Moment**

------------------------------------------------------------------------

## Overview

AetherTune is a context-aware music recommendation system that predicts
the right music based on a user's **mood, activity, and real-time
context**.

Unlike traditional platforms that rely on listening history, AetherTune
focuses on **what you need right now**.

------------------------------------------------------------------------

## Problem

Music platforms today are reactive: - Users spend time scrolling and
choosing music - Recommendations depend on past behavior - No real
understanding of current mood or activity

> Users often don't know what they want to listen to.

------------------------------------------------------------------------

##  Solution

AetherTune introduces **moment-based personalization**:

-    Mood-based recommendations\
-    Activity-aware playlists\
-    Context-driven suggestions

> The right music, at the right time---automatically.

------------------------------------------------------------------------

##  How It Works

User Input (Mood + Activity + Time)\
↓\
ML Recommendation Model\
↓\
Personalized Playlist Output
